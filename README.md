# Bienvenido a PokeFinder

Proyecto de Intercambio de cartas
-	Los usuarios van a poder conseguir diferentes cartas.
-	Los usuarios van a poder intercambiar sus cartas mediante un código QR.
-	Se podrán comprar diferentes cartas.
-	Habrá dos métodos para conseguir cartas, uno que será apareciendo cerca del usuario y una   segunda opción que es mediante sobres.
-	Habrá cartas de diferentes tipos unos siendo mas rara que otras.
-   De igual forma se podra conseguir pokemones mediante un API
